# [g-ogaki.github.io](https://g-ogaki.github.io/)

Static site for [g-ogaki.github.io](https://g-ogaki.github.io/) using:

- [Jekyll](https://jekyllrb.com/): Framework written in Ruby for building static websites
- [minimal](https://github.com/pages-themes/minimal): Official theme for GitHub Pages

## Local Development

1. Set GitHub Pages token in `.env` file
2. Set your repository in `_config.yml`
3. Run `docker compose up`
4. Open `http://localhost:4000` in your browser
